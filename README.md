## Description of Codes in this Repository
1. nfl_model_basic.ipynb -- a basic logistic regression model that aims to predict NFL game outcomes (wins/losses) using past performances and team momentum
2. nfl_model_advanced.ipynb (in development) -- a logistic regression model that aims to predict NFL game outcomes (wins/losses) in addition to spread and over/under outcomes using past performances, team momentum, and a variety of other factors (e.g., weather)
